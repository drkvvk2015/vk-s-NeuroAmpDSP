# Branding

## Release Name

Preferred public name: `NeuroAmp Live DSP`

Why this name:
- It keeps the existing NeuroAmp identity.
- It makes the live DSP use case obvious.
- It reads well in APK, Play internal testing, and release notes.

## Artifact Naming

Recommended APK filename pattern:
- `NeuroAmp-Live-DSP-v<version>+<build>.apk`

Recommended AAB filename pattern:
- `NeuroAmp-Live-DSP-v<version>+<build>.aab`

Example for the current release:
- `NeuroAmp-Live-DSP-v1.0.1+2.apk`

## Logo Direction

The logo asset in `assets/branding/neuroamp-live-dsp-logo.svg` uses:
- Charcoal base to fit audio/pro tooling.
- Emerald signal ring to imply live processing.
- Angular `N` waveform as the main mark.
- Mic capsule cutout to signal live input.

## Product Copy

Short label:
- `NeuroAmp Live DSP`

One-line description:
- `Real-time microphone and file DSP monitoring for Android.`
